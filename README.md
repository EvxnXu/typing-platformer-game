## Description

This project is a platforming game meant to teach users to type accurately and quickly and learn new words as they practice. Have Fun!!!


## How to install

Step 1. Click on the latest release under the 'Releases' tab on this page.

Step 2. Download the .exe file.

Step 3. Click on the .exe file.

Step 4. Enjoy!

## How to play

Menu - 
- Click the Button on the top right to view the leaderboard. 
- Click Play to Start a Game

In-Game - 
- As words appear on the screen, type them before time runs out!!!
- Successfully typing a word will move you to its platform and award with with extra time and points
- Hit 'Esc' to pause the game.

Game over-
- Enter your name when prompted to have your scored saved and added to the leaderboard.
